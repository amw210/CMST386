Describe how search engines work.

Search engines work by collecting all the web pages for a specific search in an algorithm within a set of high-quality results. Upon those results the user can choose which is best in that particular search. Upon the collection of searches, the search engines work by crawling hundreds of billions of pages using web crawlers. These web crawlers are commonly referred to as search engine bots or spiders. However, search engine bots are more widely used in today's society.
