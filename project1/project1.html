Describe how search engines work.

Search engines work by collecting all the web pages for a specific search in an algorithm within a set of high-quality results. Upon those results the user can choose which is best in that particular search. Upon the collection of searches, the search engines work by crawling hundreds of billions of pages using web crawlers. These web crawlers are commonly referred to as search engine bots or spiders. However, search engine bots are more widely used in today's society.

Explain the importance of keywords and descriptions as meta tags.

Keywords searches are essential because they help the user understand what they are searching for and being precise in providing to meet their needs. The keywords the user chooses give a content strategy and what to include to the website.

Meta tags do not show on the webpage itself but play a very significant role to create a webpage. Meta tags are snippets of text that describe a page's content. Meta tags only show on the page's source code. These meta tags are essentially little content descriptors that help tell the search engines what a web page is about. Meta tags only exist in HTML. It is Somewhat at the head of the page, also the word "meta." stands for metadata, which is the kind of data these tags provide.

List the four elements that comprise writing relevant content links.

  1. Avoid link text like “Click Here,” “More,” and “Read More.”
  2. Use unique link text where possible.
  3. It is OK to link a full sentence, but avoid longer.
  4. Use judgment when linking full URLs.
